# Quotes & Inspiration

Этот раздел хранит цитаты, мысли и идеи, которые помогают сохранять мотивацию и фокус.

## Как использовать

- Добавлять сюда цитаты, которые дают энергию или помогают держать курс.
- Можно записывать собственные мысли и выводы после тренировок.
- Не обязательно ждать мотивации — цель раздела в том, чтобы возвращаться к правильному настрою.

## Формат записи

```md
## Дата

> Цитата

Источник (если известен):

Почему важно:
- личная мысль
- связь с тренировками/целями
```
