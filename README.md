# Workout Log

> My Sport - My Forge

Персональный журнал тренировок, который я веду по сообщениям Streels.

## Быстрый старт

- [Дашборд](dashboard/index.html) — карточки, частота тренировок и последняя сессия.
- [Все тренировки](data/workouts.json) — единый машиночитаемый источник для дашборда.
- [Восстановление](data/recovery.json) — базовый профиль сна и восстановления.
- [Активность](data/activity.json) — шаги и повседневная нагрузка.
- [Тренировки](workouts/) — отдельная Markdown-запись на каждую тренировку.

## Структура

```text
.
├── README.md
├── data/
│   ├── workouts.json         # Тренировки и нагрузки
│   ├── recovery.json         # Сон и восстановление
│   └── activity.json         # Шаги и активность
├── workouts/
│   └── 2026/07/              # Отдельные тренировки по датам
│       └── 17-back-shoulders-core.md
└── dashboard/
    └── index.html            # Автономный веб-дашборд
```

## Как ведём журнал

Streels присылает упражнения, вес, подходы/повторы и ощущения после тренировки. Я создаю отдельную запись, обновляю `data/workouts.json` и дашборд.
