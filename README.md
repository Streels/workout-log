# Workout Log

> My Sport - My Forge

Персональный журнал тренировок Streels. Журнал ведётся совместно с ChatGPT: анализ прогресса, планирование тренировок и обновление данных выполняются на основе сообщений после каждой тренировки.

## Быстрый старт

- [Дашборд](dashboard/index.html) — карточки, частота тренировок и последняя сессия.
- [Все тренировки](data/workouts.json) — единый машиночитаемый источник для анализа прогресса.
- [Восстановление](data/recovery.json) — сон и восстановление.
- [Активность](data/activity.json) — шаги и повседневная нагрузка.
- [Питание и добавки](data/nutrition.json) — текущий режим питания и БАДы.
- [Тренировки](workouts/) — отдельные Markdown-записи каждой тренировки.
- [Замеры тела](progress/body_measurements/) — контрольные точки формы и рекомпозиции.

## Цель

Главная цель: рекомпозиция тела — увеличение мышечной массы с одновременным улучшением формы и более выраженным прессом.

## Принципы ведения тренировочного процесса

### 1. Прогрессия нагрузок

Основной принцип — постепенное увеличение нагрузки.

ChatGPT отслеживает:
- рабочие веса;
- количество повторений;
- количество подходов;
- сложность выполнения;
- субъективные ощущения после тренировки.

При готовности организма увеличивается вес, количество повторений или объём работы. Если нагрузка становится чрезмерной — корректируется в обратную сторону.

### 2. Обновление тренировочного плана

Тренировочная программа не является постоянной.

Примерно раз в месяц план пересматривается:
- меняются упражнения;
- корректируется объём;
- добавляются новые варианты нагрузки;
- учитывается прогресс и слабые места.

Цель — сохранять прогресс, избегать привыкания и развивать тело комплексно.

### 3. Контроль формы

Streels периодически отправляет фотографии формы.

ChatGPT самостоятельно напоминает о необходимости сделать новые фотографии для оценки прогресса.

Оцениваются:
- изменение мышечной массы;
- процент жира визуально;
- пропорции тела;
- изменения относительно предыдущих периодов.

### 4. Анализ восстановления

При планировании тренировок учитываются:
- качество сна;
- уровень энергии;
- шаги и общая активность;
- субъективное состояние.

## Как ведём журнал

Перед каждой новой тренировкой:

1. Анализируется история тренировок.
2. Проверяются предыдущие рабочие веса.
3. Учитывается восстановление.
4. Формируется новый план с учётом прогрессии.

После тренировки Streels отправляет отчёт:
- упражнения;
- веса;
- подходы и повторения;
- ощущения.

ChatGPT:
- анализирует тренировку;
- фиксирует результат;
- обновляет журнал;
- корректирует дальнейшую прогрессию.
